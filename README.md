🏡 Airbnb Home Page Clone
A pixel-inspired clone of the Airbnb home page, built with plain HTML, CSS, and Tailwind CSS (via CDN). This project focuses solely on the frontend, recreating the look, feel, and responsiveness of Airbnb's landing page using utility-first styling, with no build tools or frameworks.

📌 Project Overview
This project is a frontend-only clone of the Airbnb home page. The goal was to practice building a clean, responsive UI using plain HTML and Tailwind CSS, closely matching Airbnb's real-world design patterns such as navigation bar, search bar, category filters, and property listing cards.

There is no backend, authentication, database, or JavaScript framework involved. Tailwind is loaded directly via CDN, so there's no build step required — open the HTML file, and it works.

✨ Features
Responsive Layout — adapts smoothly across desktop, tablet, and mobile screens
Airbnb-Style Navbar — logo, search bar, and menu matching the original design
Category Filters — icon-based filter bar for browsing property types
Property Listing Cards — image, title, price, and rating layout similar to Airbnb
Utility-First Styling — built entirely with Tailwind CSS classes, no custom CSS files
Zero Build Setup — Tailwind via CDN means no npm install or bundler needed

🛠️ Tech Stack
Layer	Technology

⚙️ Setup Instructions
Prerequisites
A modern web browser
That's it — no Node.js or npm required

🎨 Running the Project
1. Clone the repository
git clone https://github.com/Shafin98/Airbnb-Home-Page.git
cd Airbnb-Home-Page

2. Open the project
Simply open index.html directly in your browser, or use a live server for auto-reload during development:
Using VS Code: right-click index.html → "Open with Live Server"
Or just double-click index.html to open it in your default browser
